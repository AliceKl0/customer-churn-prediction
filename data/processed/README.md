# Обработанные данные (Processed Data)

Эта директория содержит обработанные и подготовленные данные, готовые для обучения моделей.

## Описание

После обработки исходных данных здесь сохраняются:
- Очищенные данные (без пропусков, выбросов)
- Закодированные категориальные переменные
- Масштабированные числовые признаки
- Разделенные на train/test наборы данных

## Формат

Обработанные данные обычно сохраняются в формате:
- CSV для просмотра
- Pickle (.pkl) для быстрой загрузки в Python
- Parquet для эффективного хранения больших объемов

## Автоматическая обработка

Проект автоматически обрабатывает данные при запуске `main.py`. Обработанные данные могут быть сохранены здесь для последующего использования без повторной обработки.
