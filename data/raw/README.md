# Исходные данные (Raw Data)

Эта директория содержит исходные, необработанные данные проекта.

## Инструкции

1. Поместите исходные CSV файлы с данными клиентов в эту директорию
2. Файлы должны содержать следующие колонки (или аналогичные):
   - `customer_id` - уникальный идентификатор клиента
   - `credit_score` - кредитный рейтинг
   - `age` - возраст клиента
   - `tenure` - количество лет сотрудничества с банком
   - `balance` - баланс счета
   - `num_of_products` - количество продуктов банка
   - `has_cr_card` - наличие кредитной карты (0/1)
   - `is_active_member` - активность клиента (0/1)
   - `estimated_salary` - предполагаемая зарплата
   - `exited` - целевая переменная: ушел ли клиент (0/1)
   - `geography` - страна
   - `gender` - пол

3. Если данных нет, проект автоматически сгенерирует синтетические данные для демонстрации

## Формат данных

- Формат: CSV
- Кодировка: UTF-8
- Разделитель: запятая

## Примечание

Если вы используете реальные данные, убедитесь, что они соответствуют требованиям конфиденциальности и защиты персональных данных.
